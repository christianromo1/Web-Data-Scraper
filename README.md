# Web Data Scraper

A Python-based data scraping tool designed to collect structured data from websites for machine learning training datasets.

## Features
- Automated web scraping using Python
- Data extraction and cleaning
- Structured dataset generation for ML model training
- Configurable scraping targets

## Technologies Used
- Python
- Requests / BeautifulSoup (or whatever you used)
- Data processing libraries

## Purpose

This project was developed as part of a university assignment focused on collecting datasets to support machine learning model training.
